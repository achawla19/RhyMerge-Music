# 🎵 RhyMerge

A modern platform for music creators to **connect, collaborate, and create together**.

---

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS
- Lucide Icons

---

## 🚀 Getting Started

```bash
git clone https://github.com/achawla19/RhyMerge-Music.git
cd rhymerge
npm install
npm run dev
```
